# Changelog

## [2026-04-18] Цветовая индикация календаря v2

### Добавлено  
- Улучшенная цветовая индикация дней в календаре на основе реальной доступности:
  - 🟢 Зелёный круг - дни с доступными свободными слотами
  - 🟡 Жёлтый круг - дни где все слоты заняты
  - 🔴 Красный круг - закрытые дни или дни без слотов

### Изменения
- Добавлена функция `get_days_availability_for_period()` в `database/db.py`
- Обновлена логика `build_calendar()` для отображения реальной доступности
- Календарь теперь показывает не просто открытые/закрытые дни, а наличие свободных мест

### Техническая информация
- Оптимизированный запрос с JOIN для получения статуса слотов и бронирований
- Три состояния: 'available', 'full', 'closed'
