# Changelog

## [2026-04-18] Цветовая индикация календаря

### Добавлено
- Цветовая индикация дней в календаре:
  - 🟢 Зелёный круг - рабочие дни (доступные для записи)
  - 🔴 Красный круг - нерабочие дни (закрытые для записи)

### Изменения
- Обновлена функция `build_calendar()` в `keyboards.py`
- Добавлена функция `get_days_status_for_period()` в `database/db.py`
- Рефакторинг кода календаря для улучшения читаемости

### Техническая информация
- Календарь теперь запрашивает статус всех дней месяца одним запросом к БД
- Эмодзи-индикаторы добавляются перед номером дня в кнопках календаря
